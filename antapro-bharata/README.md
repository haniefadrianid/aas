## ⚡ Optimasi Performance

### Core Web Vitals Strategy

#### LCP (<2s)
- ✅ Critical CSS inline
- ✅ Hero image preload
- ✅ Font preload
- ✅ WebP format
- ⚠️ Ganti placeholder images dengan WebP asli

#### INP (<200ms)
- ✅ Vanilla JavaScript (no jQuery)
- ✅ Deferred scripts
- ✅ Minimal DOM manipulation
- ✅ Passive event listeners

#### CLS (<0.1)
- ✅ Explicit width & height pada gambar
- ✅ No dynamic content injection
- ✅ Font-display: swap

### Checklist Performance
- [ ] Compress images (TinyPNG/Squoosh)
- [ ] Convert to WebP format
- [ ] Minify HTML (optional)
- [ ] Enable Gzip/Brotli compression
- [ ] Add caching headers
- [ ] Use CDN for images

## 📈 Scaling Campaign Strategy

### Fase 1: Launch (Minggu 1-2)
**Budget:** Rp 500.000 - 1.000.000 / hari

**Keyword Strategy:**
- 5-10 long-tail keywords
- Exact match & Phrase match
- Contoh:
  - "obat stroke herbal bpom"
  - "herbal untuk pemulihan stroke"
  - "antapro bharata"

**Action:**
- Pantau Quality Score harian
- Pause keyword CTR < 1%
- Tambahkan negative keywords

---

### Fase 2: Optimasi (Minggu 3-4)
**Budget:** Scale 20-30% jika CVR > 3%

**Action:**
- Duplicate ad group dengan match type berbeda
- Tambahkan RSA (Responsive Search Ads) dengan 15 headlines
- A/B test 2 versi iklan
- Tambahkan ad extensions lengkap

---

### Fase 3: Scale (Bulan 2)
**Budget:** Scale 50% jika ROI positif

**Action:**
- SKAG (Single Keyword Ad Group) untuk top performer
- Tambahkan Display Remarketing
- YouTube ads dengan video testimoni
- Lookalike audience dari converters

---

### Fase 4: Expansion (Bulan 3+)
**Budget:** Unlimited (jika ROI positif)

**Action:**
- Performance Max Campaign
- Shopping Ads (jika produk fisik)
- Expand ke keyword broader
- Geographic expansion
